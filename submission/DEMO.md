# デモを試す手順（営業支援エージェント）

営業の初回・継続の提案資料づくりを、過去の面談データを引き継いで自走する AI エージェントです。
初回はアポ資料、2回目以降は前回実績を踏まえた継続提案資料を自動生成します。

## アクセス情報

- **URL**: https://ca-frontend.delightfulpebble-6aae50fe.eastus2.azurecontainerapps.io/
- **アクセスコード**: チームに別途共有します（Slack 等）。
  > ⚠️ アクセスコードは公開記事・SNS・公開リポジトリには記載しないこと。

## 使い方

1. 上記 URL を開く（初回はコールドスタートのため数秒待つ）。
2. 「アクセスコード」に共有されたコードを入力して「入室」。
3. 会社名・業種などを入力し「🚀 資料を生成」→ 生成された PowerPoint をダウンロード。
4. **2回目以降（継続提案）を試す場合**：
   面談ステータスを「2回目以降」にし、会社名に既存客 `株式会社既存お得意様` を入れて
   「前回面談メモ」を記入 → 前回履歴を踏まえた継続提案が生成されます。

## 注意事項

- 投入データは**ダミー / シードのみ**です（実顧客データは入れないこと）。
- 連続実行には約 15 秒のクールダウンがあります。
- 生成すると面談レコードが Cosmos DB に追加されます（テスト後は適宜削除）。
- 審査期間（2026/6/2〜6/18）はこの環境を稼働維持します。

## 技術スタック

- 実行基盤: Azure Container Apps（API＋フロント）
- AI: Azure OpenAI (Microsoft Foundry) gpt-4.1 ＋ Semantic Kernel
- RAG: Azure Cosmos DB Vector Search
- 認証: Microsoft Entra ID（Managed Identity）／ストレージ: Azure Blob Storage
